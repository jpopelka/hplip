Installation - Step 1 - APT

  1. Open a console/terminal window.
  2. Follow these steps to log in as the super user:
  3. Enter this command:
su
  1. Press Enter.
  2. Enter the root password.
  3. For Debian and Ubuntu based distributions, enter this command:
dpkg --list 'hplip*'
  1. You should see something like this displayed:
Name       Version        Description
====       =======        ===========
hplip      (none)         (no description available)
hplip-base 0.9.5-2ubuntu2 hplip - base system
hplip-data 0.9.5-2ubuntu2 hplip - data files
hplip-ppds 0.9.5-2ubuntu2 hplip - PPD files

If "No Packaged Found matching "hplip*" is displayed, HPLIP is not installed.

If HPLIP is installed follow the steps under Step 4: Setup your printer, if HPLIP is not installed follow the steps under Step 2: Set up the Linux environment.

Proceed to Step 2.
