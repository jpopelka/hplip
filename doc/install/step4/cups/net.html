Installation - Step 4 - CUPS - Network

  1. Connect the printer to the network.

Note: See your printer documentation for specific instructions on connecting the USB, Parallel or Network cable to your printer and computer.

  1. Using the Printer Setup Menus on your printer, print the "Network Configuration" page. This page will have the IP address of the printer, which is essential to complete the printer setup.

Note: See your printer documentation for specific instructions on printing the network configuration page.

  1. Open a console/terminal window.
  2. Enter this command to search for the printer URI: (Example: "hp-makeuri 192.168.1.100")
hp-makeuri <printer.IP.address.from.step.2>
  1. Copy the CUPS URI created for the printer. (Example: "hp:/net/HP_Business_Inkjet_1200?ip=192.168.1.100")

Hint: Use the right-click menu to copy and paste text.

  1. Open a web browser, and then enter the following address in the Address box: http://localhost:631
  2. On the menu bar at the top of the page, click Printers -> Add Printer
  1. Enter the root username and password. (when prompted)
  2. Enter a name for your HP printer in the Name box.

Note: The name you enter must not contain spaces.

  1. (Optional) Enter a description of the location of the printer in the Location box. (Example: 3rd Floor Office)
  2. (Optional) Enter a description of the printer in the Description box. (Example: HPLJ3380)
  3. Click Continue.
  4. From the Device drop-down list, click Appsocket/JetDirect, and then click Continue.
  1. In the Device URI text box, paste the printer URI you copied in step 5, and then click Continue.
  1. In the Make list, click HP -> Continue.
  1. In the Model list, click the printer model that most closely matches your printer, and then click Continue.

Note: To configure your printer for using Fax, follow these steps, otherwise go to Step 28.

  1. Once the printer is added using the above steps, go back to step 6.
  2. At step 9, enter a name for your Fax device. Example: "photosmart_3310_fax"
  3. At step 14, select your device that starts with "hpfax://". Example: "hpfax:/usb/psc_2500_series?serial=XXXXXXXXX (HP psc_2500_series)
  4. At step 15, select HP
  5. At step 13, scroll to the very bottom and select "HPLIP Fax (en)
  6. Complete the setup process.
  7. Open the hp-toolbox. (Open a terminal/console window and enter;)

Note: You can also open the HP Device Manager by using the GNOME or KDE menu system.

hp-toolbox

Note If setting up fax continue else go to step 30.

  1. Select your printer.
  2. Click the "Tools & Settings" tab.
  3. Click "Setup Fax.."
  4. Enter your information in the "Fax Header Information" section.
  5. Click "Ok".
  6. Fax setup is now complete.
  1. Click the Tools & Settings tab, and then click Print Test Page.
