Installation - Step 4 - hp-setup - Network

Note: See your printer documentation for specific instructions on connecting the USB, Parallel or Network cable to your printer and computer.

  1. Open a console/terminal window.
  2. Follow these steps to log in as the super user; (if using Ubuntu/Debian goto step 7)
  3. Enter this command:

Warning: The root password gives you administrative privileges on the system.

su
  1. Press Enter.
  2. Enter the root password.
  3. Press Enter.
  4. Enter this command to add your USB or Parallel printer: (You can also use the -a option to setup your printer with a default name and settings.)
hp-setup <ip address of printer>

or to perform automatic setup:

hp-setup -a <ip address of printer>

for Ubuntu enter:

sudo hp-setup -a <ip address of printer>
  1. Follow the steps as prompted to complete the printer setup.

Note: You may see a message stating "[WARNING]: Found multiple possible PPD files"--If prompted, select the option that best fits your printer model.

  1. Your printer should print out a test page. If the test page prints successfully, your printer is configured correctly.
  2. If your printer supports faxing you will want to complete the fax configuration as prompted.
  3. Printer setup is now complete.
